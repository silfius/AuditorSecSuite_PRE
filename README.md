# AuditorSecSuite_PRE

Suite interna de apoyo a auditorías técnicas de seguridad.

No es producto vendible ni SaaS. Uso previsto: personal e interno de empresa.

## Alcance inicial

- Inventario de activos.
- Auditorías controladas.
- Findings.
- Evidencias.
- Informes internos.
- Integración futura con motores existentes: testssl/SSLyze, Nuclei, OWASP ZAP y Greenbone CE.

## Regla principal

Solo activos propios o con autorización explícita.
