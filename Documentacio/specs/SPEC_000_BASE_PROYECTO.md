# SPEC_000_BASE_PROYECTO — AuditorSecSuite_PRE

## Objetivo
Crear la base del proyecto con runtime Docker, Django, PostgreSQL, documentación viva, modelos iniciales y validadores.

## Alcance
Scaffold, modelos Activo/Auditoria/Finding, vistas mínimas, tests unitarios y scripts.

## Fuera de alcance
Escaneos reales, Greenbone, ZAP, Nuclei, testssl, informes PDF/DOCX y publicación externa.

## Riesgos
Versionar secretos, exponer una herramienta dual-use, ejecutar pruebas contra activos no autorizados.

## Criterios de aceptación
Estructura creada, documentación viva presente, compose válido, scripts ejecutables y sin secretos versionables.
